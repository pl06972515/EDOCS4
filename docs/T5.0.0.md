!> <span style='color:red'>在`TS`中，接口常用于描述对象 [ 对象注解 - 限制对象结构 ]</span>

```csharp
interface IClass {

     x: number
     y: number
	
     M1(x: number, y: number): void
    
}


```

```cs
class TClass implements IBDrawable {
    
     x: number;
     y: number;

     constructor(x: number, y: number) {
         this.x = x;
         this.y = y;
     }
     
     M1(): void {
     
         throw new Error("Method not implemented.");
     } 

}


```

#### **[ ℹ️ 常用属性 ]**

```csharp
# [ 平台检测 ]
Boolean(Ext.isLinux)   // 检测是否在 Linux 系统上运行
Boolean(Ext.isWindows) // 检测是否在 Windows 系统上运行
Boolean(Ext.isMac)     // 检测是否在 Mac 系统上运行

Boolean(Ext.isIE)      // 检测是否在 Internet Explorer 浏览器中运行
Boolean(Ext.isFirefox) // 检测是否在 Firefox 浏览器中运行
Boolean(Ext.isChrome)  // 检测是否在 Chrome 浏览器中运行
Boolean(Ext.isSafari)  // 检测是否在 Safari 浏览器中运行
    
# [ 版本信息 ]
Ext.versions.core         // 核心库版本信息
Ext.versions.core.version // 核心库版本号字符串


```



#### **[ ✡ 常用方法 ]**

```csharp
Ext.getApplication() // Ext.app.Application

Ext.getCmp(id)       // 查找组件 ( Ext.Component )
Ext.getStore(name)   // 数据存储 ( Ext.data.Store )

----------------------------------------
    
Ext.getClass(object)        // 获取类定义 ( Ext.Class )
Ext.getClassName(object)    // 获取类名称
Ext.getNamespace(className) // 根据类名获取 [ 命名空间前缀 ]

    
```

```csharp
# [ 类型检查 ]
Ext.isPrimitive(target) // [ 检测 ] 是否简单类型 ( 如：string、number、boolean )
Ext.isNumber(target)  
Ext.isString(target)
Ext.isBoolean(target)
Ext.isDate(target)
    
Ext.isObject(target)：
Ext.isArray(target) 
Ext.isFunction(target)
    
Ext.isEmpty(target)    // 检查值是否为 ( null, undefined, 长度为 0 的数组, 长度为 0 的字符串 )
Ext.isIterable(target) // 是否允许迭代

-------------------------------------
    
# [ 统一遍历 ] 对象或数组
// 遍历对象
Ext.iterate({ name: 'John', age: 30 }, function(key, value) {
    console.log(key + ': ' + value);
});

// 遍历数组
Ext.iterate([1, 2, 3], function(value, index) {
    console.log(index + ': ' + value);
});


```

```csharp
# [ 获取 DOM 元素 ] Ext.dom.Element
Ext.get(element) 
Ext.getWin()     // 获取 window
Ext.getHead()    // 获取 head
Ext.getBody()    // 获取 body
Ext.getDoc()     // 获取 document

Ext.getViewportHeight() // 获取窗口视口高度
Ext.getViewportWidth()  // 获取窗口视口宽度
  
# Ext.dom.Element.query ( 根据CSS选择器选择DOM元素 - 类似 jQuery )
Ext.query(selector, [asDom]) // HTMLElement[] | Ext.dom.Element[]
# Ext.dom.Element.select
Ext.select(selector, composite)
   
```

#### **[ 配置文件 ] tsconfig.json**

```csharp
E:\VisualStudioCode\pm2.html.test\    <-- 项目根目录
├── tsconfig.json          # 基础配置( 全局 )
├── apps/
│   └── test-app/
│       └── tsconfig.json  # 子包配置( 继承 + 覆盖 )
│
└── packages/
    └── some-lib/
        └── tsconfig.json  # 子包配置( 继承 + 覆盖 )
    
    
```

```csharp
# [ tsconfig.json ] [ 全局配置 ]
{
    
     "compilerOptions": {
          
          "noEmitOnError": true, // [ 阻止编译 ] 当 .ts 因代码错误，禁止编译为 .js
              .
              .
              .
           
     }
     
}


```

```csharp
# [ tsconfig.json ] [ 子包配置 ]
{
      
     # [ A ][ 继承根目录配置 ]
     "extends": "../../tsconfig.json",
    
     # [ B ][ 覆盖根目录配置 ]
     "compilerOptions": {
          
          "outDir": "./dist" // [ 输出路径 ] 编译文件输出路径
          
      },
     
     "include": [
          "./**/*.ts"
      ]
      
}


```



#### **[ ℹ️ 编译命令 ]**

>[!WARNING|style: flat|label: 简要说明 ]
>
>- <span style='color:red'>[ 编译文件`.ts`：`pnpm exec tsc | pnpm exec tsc xxx.ts`] 将`TypeScript`代码编译转换为`.js`</span>
>
>- <span style='color:Blue'>[ 即时编译`.ts`：`pnpm exec ts-node xxx.ts`] 它在内存中即时编译`TypeScript`代码 [ 然后立即执行 ]</span>
>
>  <span style='color:Blue'>[ 项目根目录安装`pnpm add -D ts-node -w`]</span>
>
><br/>

```csharp
# [ 配置 vscode ] settings.json ( 插件 code-runner: 在执行 .ts 时需运行的命令 )
"code-runner.executorMap": {
     "typescript": "cd $dir && pnpm exec ts-node $fileName"
 }


```

#### **[ 配置文件 ] tsconfig.json**

!> <span style='color:Blue'>[`pnpm exec tsc --init`]</span>

```csharp
E:\VisualStudioCode\pm2.html.test\    <-- 项目根目录
├── tsconfig.json          # 基础配置( 全局 )
├── apps/
│   └── test-app/
│       └── tsconfig.json  # 子包配置( 继承 + 覆盖 )
│
└── packages/
    └── some-lib/
        └── tsconfig.json  # 子包配置( 继承 + 覆盖 )
    
    
```

```csharp
# [ tsconfig.json ] [ 全局配置 ]
{
    
     "compilerOptions": {
          
          "noEmitOnError": true, // [ 阻止编译 ] 当 .ts 因代码错误，禁止编译为 .js
              .
              .
              .
           
     }
     
}


```

```csharp
# [ tsconfig.json ] [ 子包配置 ]
{
      
     # [ A ][ 继承根目录配置 ]
     "extends": "../../tsconfig.json",
    
     # [ B ][ 覆盖根目录配置 ]
     "compilerOptions": {
          
          "outDir": "./dist" // [ 输出路径 ] 编译文件输出路径
          
      },
     
     "include": [
          "./**/*.ts"
      ]
      
}


```



#### **[ ℹ️ 编译命令 ]**

>[!WARNING|style: flat|label: 简要说明 ]
>
>- [ 编译文件`.ts`：`pnpm exec tsc | pnpm exec tsc xxx.ts`] 将`.ts`文件编译转换`.js`
>
>- <span style='color:RED'>[ 运行脚本`.ts`：`pnpm exec tsx $dir/xxx.ts`] 直接运行 .ts 文件 ( 跳过编译 - 需相对路径 )</span>
>
>  <span style='color:RED'>[ 项目根目录安装`pnpm add -D tsx -w`]</span>
>
><br/>

```csharp
# [ 配置插件 code-runner ] settings.json ( vscode )
"code-runner.executorMap": {
     "typescript": "pnpm exec tsx $dir/$fileName"
 }


```
